# Circles for Kinect

A customisable Kinect MIDI controller for Windows, designed to allow musicians to create custom Kinect instruments without needing any knowledge of programming. 

[calhewitt.github.io/circles-kinect](http://calhewitt.github.io/circles-kinect)

##Building and Requirements 

Circles is developed in and can be built in Visual Studio (express) and requires [version 1 of the Kinect Windows SDK](http://www.microsoft.com/en-us/download/details.aspx?id=40278). Once built, the software only needs the Kinect runtime to run.
